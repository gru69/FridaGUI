## 📥 Installation

### Prerequisites
- **Python 3.8+**
- **Frida**
- **ADB** (for Android device support)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/oliverstankiewicz/FridaGUI.git
   cd FridaGUI
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python src/main.py
   ```
